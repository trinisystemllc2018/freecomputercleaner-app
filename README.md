# Trini System LLC — Tech Support Website

A modern React website for **Trini System LLC**, rebuilt from Google Sites with a professional dark-tech design.

## 🚀 Deploy to Vercel (Free)

### Option 1: GitHub + Vercel (Recommended)

1. **Create a GitHub repo** at https://github.com/new
2. **Push this code:**
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Trini System LLC website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/trini-system-llc.git
   git push -u origin main
   ```
3. **Deploy on Vercel:**
   - Go to https://vercel.com → Sign up free with GitHub
   - Click **"Add New Project"**
   - Import your `trini-system-llc` repo
   - Framework: **Create React App** (auto-detected)
   - Click **Deploy** ✅

4. **Your free URL:** `https://trini-system-llc.vercel.app`

### Option 2: Vercel CLI (Direct Deploy)

```bash
npm install -g vercel
vercel login
vercel --prod
```
